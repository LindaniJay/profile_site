Building My Modern Portfolio Website: A Deep Dive

A behind-the-scenes look at designing, developing, and modernizing my interactive portfolio and weather dashboard.

Project Vision

  • Visually striking dark theme with navy, black, and deep accent colors
  • Highly interactive: live weather dashboard, animated backgrounds, smooth transitions
  • Professional & personal: formal intro, detailed about, portfolio of latest projects
  • Responsive & accessible for all devices

Key Features

  • Modern Dark Theme: Custom palette, gradients, glassmorphism, and video backgrounds
  • Sidebar Navigation: Fixed, collapsible, with icons and live clock
  • Home & About: Formal intro, skills, and a modern card layout
  • Portfolio Showcase: Latest projects in stylish cards
  • Weather Dashboard: Real-time data, animated backgrounds, 5-day forecast, geolocation, robust error handling
  • Blog & Contact: Technical articles and a modern contact form
  • Game Page: A fun, interactive JavaScript game

Tech Stack

  • HTML5: Semantic, accessible markup
  • CSS3: Custom properties, Flexbox, Grid, glassmorphism, gradients, and video backgrounds
  • Vanilla JavaScript (ES6+): All interactivity, API integration, geolocation, and animations (no React or frameworks)
  • OpenWeatherMap API: Real-time weather data (current, hourly, 5-day)
  • FontAwesome: Crisp, scalable icons
  • Responsive Design: Media queries and flexible layouts for all devices
  • Assets: Custom images, profile photos, and a looping background video

Debugging & Modernization

  • Checked all image/script links, set up linter, improved accessibility, and optimized performance
  • Removed outdated elements, unified all pages, and added animated/video backgrounds

Challenges & Solutions

  • Weather API integration: geolocation, error handling, and data formatting
  • Animated backgrounds: balancing visual appeal with readability and performance
  • Responsiveness: ensuring a great look on all devices

Final Thoughts

This project blends design, interactivity, and real-world data into a single, cohesive portfolio. The result is a site that not only showcases my work but also demonstrates my ability to build modern, user-friendly web applications.

Check it out, explore the weather dashboard, and let me know what you think!
